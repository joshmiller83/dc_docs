Add-to-Cart Buttons for Drupal Commerce

What is the Add-to-Cart button?

Almost everything in Drupal Commerce is put together using a common Drupal pattern: Content (entity and fields system), Views (listing things), and Rules (taking actions on events). The Add-to-Cart button is no different. But there is a bit of mythical Drupal magic that makes this button show up and we aim to help you peer behind the curtains.

Add-to-Cart Configuration using Fields

With Drupal Commerce 1.2, we've replaced the Add-to-Cart magic with dependable configuration settings. By altering forms in the Field UI, we now embed options to help you create complex Add to Cart forms. Below are three documented ways to access, alter, or create Add-to-Cart buttons.

  1. Product Reference Field - An Add-to-Cart button that is configured on the Display Node.
  2. Product Type Field - An Add-to-Cart button that is configured on the Product Entity level.
  3. Line Item Field - An Add-to-Cart button that is configured on the Line Item level.

Add-to-Cart Buttons using Product Reference Fields

[img]

When you add a product reference field to an entity, you have the option of injecting fields from the referenced products into the display of the referencing entity.

A great example of a product reference field is the one found in the Product display node type that Commerce Kickstart creates. The product reference field enables prices, images, and other fields attached to your products to be displayed in the node along with an Add-to-Cart button.

Add-to-Cart Buttons using Product Entity Fields

[img] Caption

By adding fields to your product types, you can create a group of products representing variations of the same base product (e.g. different sizes of the same style and design of a shirt). There's a simple checkbox to start using field data on the Add to Cart form along with an optional widget choice.

Line Items, Field Configuration

[img] Caption

By adding fields to your line item types, you can expose customization options to the customer on the Add to Cart form. If you have multiple types of customizable products, you can create additional product line item types and use them on different Add to Cart forms.
